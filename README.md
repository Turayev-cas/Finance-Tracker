## Summary  
A simple desktop Python application for tracking personal income and expenses with a tkinter GUI. **Context:** Built as a learning project and a portfolio piece to support my application to a Financial Technology program in China.

- ‚úÖ Tracks balance by adding incomes and expenses
- ‚úÖ Stores transaction history in `data.json`

## üìñ How to Use

### Quick steps

1. Enter an amount in the input field
2. Click **‚ûï Add income** to add income or **‚ûñ Add Expense** to add an expense
3. Toggle **Show history** to view transactions
4. Click **Edit**, modify the history text, then **Save** to recalculate the balance
